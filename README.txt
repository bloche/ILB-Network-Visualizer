This project was created for Software Engineering I at the University of Idaho, Moscow, ID by team Intentionally Left Blank
All code is open-source and was created for educational purposes.

This software is intended to be used as a graphing and analysis tool for virtual networks, though it can be used for any network
that can be specified in an xml file of the required format.

Documentation for this project can be found at http://www2.cs.uidaho.edu/~cs383c
